# Airport Findr

# Stack
- Next.js 15
- Next Server Actions
- Tailwind CSS
- Shadcn UI
- Leaflet

# Q & A
How much time was spent developing it: 11 hours



# How to run
1. Create an `.env` file with the following variables, see `.env.example` for reference
    - AIRPORT_API_USERNAME=
    - AIRPORT_API_PASSWORD=
2. npm install
3. npm run dev
